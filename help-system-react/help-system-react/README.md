# Help System React

Projeto React em JavaScript convertido a partir das páginas HTML/JS originais do Help System.

## Como rodar

```bash
npm install
npm run dev
```

## Stack

- React + Vite
- React Router DOM
- Lucide React
- Chart.js + react-chartjs-2
- localStorage/sessionStorage para persistência local

## Estrutura principal

- `src/pages`: telas da aplicação
- `src/components`: componentes reutilizáveis
- `src/services`: lógica de autenticação, storage e regras de negócio
- `src/context`: estado global de autenticação
- `src/styles/globals.css`: estilos globais
